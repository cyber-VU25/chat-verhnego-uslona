# ЧАТ ВЕРХНЕГО УСЛОНА

Готовый сайт-чата для участников с регистрацией по номеру телефона и управлением через Telegram-бота.

## Что внутри

- регистрация и вход по номеру телефона через одноразовый код;
- общий чат в реальном времени;
- лёгкий адаптивный дизайн;
- SQLite-база без отдельного сервера БД;
- Telegram-команды администратора:
  - `/users` — список пользователей;
  - `/ban +79990000000` — заблокировать;
  - `/unban +79990000000` — разблокировать;
  - `/broadcast текст` — отправить объявление в чат;
  - `/help` — помощь.

## Быстрый запуск

1. Установите Node.js 20+.
2. Скопируйте `.env.example` в `.env`.
3. Установите зависимости:

```bash
npm install
```

4. Запустите:

```bash
npm start
```

5. Откройте сайт:

```text
http://localhost:3000
```

В демо-режиме код входа будет печататься в консоли сервера.

## Подключение Telegram-управления

1. В Telegram откройте `@BotFather`.
2. Создайте бота командой `/newbot`.
3. Вставьте токен в `.env` как `TELEGRAM_BOT_TOKEN`.
4. Напишите своему боту любое сообщение.
5. Узнайте свой `chat_id`, например через `@userinfobot`, и вставьте его в `TELEGRAM_ADMIN_CHAT_ID`.
6. Перезапустите сервер.

## Подключение настоящих SMS

Сейчас включён `OTP_MODE=demo`: код печатается в консоль.  
Для настоящих SMS замените функцию `sendSmsCode(phone, code)` в `server.js` на вызов вашего SMS-провайдера.

## Продакшн

Для публикации на сервере:
- задайте длинный `SESSION_SECRET`;
- включите HTTPS;
- подключите SMS-провайдера;
- запускайте через PM2, Docker или systemd;
- сделайте регулярный бэкап файла `data/chat.sqlite`.
